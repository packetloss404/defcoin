# Security Policy

## Reporting a Vulnerability

To report security issues, please open a private security advisory at:
https://github.com/packetloss404/defcoinv2/security/advisories/new

Alternatively, you can open an issue at:
https://github.com/packetloss404/defcoinv2/issues

Please do NOT disclose security vulnerabilities publicly until they have been
addressed by the maintainers.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | Yes                |
| < 2.0   | No                 |

## Security Best Practices

When running Defcoin Core:

1. Keep your wallet encrypted with a strong passphrase
2. Backup your `wallet.dat` file regularly
3. Run the latest version of Defcoin Core
4. Use a firewall to restrict RPC access
5. Never share your private keys or wallet file
